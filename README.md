**convertRGBtoCMYK(Color color):**
–ü—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç —Ü–≤–µ—Ç –∏–∑ RGB –≤ CMYK.  
–ü—É—Å—Ç—å R, G –∏ B ‚Äî —ç—Ç–æ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã —Ü–≤–µ—Ç–∞ –≤ –º–æ–¥–µ–ª–∏ RGB, –Ω–∞—Ö–æ–¥—è—â–∏–µ—Å—è –≤ –¥–∏–∞–ø–∞–∑–æ–Ω–µ –æ—Ç 0.0 –¥–æ 1.0 (–∞ –Ω–µ –æ—Ç 0 –¥–æ 255!). –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ —Ä–∞–∑–¥–µ–ª–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è –≤ –æ–±—ä–µ–∫—Ç–µ Color –Ω–∞ 255.0. –ü–æ—Å–ª–µ —ç—Ç–æ–≥–æ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

–ï—Å–ª–∏ –Ω–µ—Ç, —Ç–æ —Å–ª–µ–¥—É—é—â–∏–µ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è:

**convertRGBtoHSV(Color color):**
–ü—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç —Ü–≤–µ—Ç –∏–∑ RGB –≤ HSV. 
–ü—É—Å—Ç—å MAX –±—É–¥–µ—Ç –º–∞–∫—Å–∏–º–∞–ª—å–Ω—ã–º, –∞ MIN ‚Äî –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–º –∑–Ω–∞—á–µ–Ω–∏–µ–º —Å—Ä–µ–¥–∏ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–≤ (R, G, B). –ó–Ω–∞—á–µ–Ω–∏–µ H (–æ—Ç—Ç–µ–Ω–æ–∫) –¥–æ–ª–∂–Ω–æ —É–¥–æ–≤–ª–µ—Ç–≤–æ—Ä—è—Ç—å —É—Å–ª–æ–≤–∏—é 0 ‚â§ H < 360, –∞ –∑–Ω–∞—á–µ–Ω–∏—è S (–Ω–∞—Å—ã—â–µ–Ω–Ω–æ—Å—Ç—å), V (—è—Ä–∫–æ—Å—Ç—å), R, G –∏ B –±—É–¥—É—Ç –Ω–∞—Ö–æ–¥–∏—Ç—å—Å—è –≤ –¥–∏–∞–ø–∞–∑–æ–Ω–µ 0..1.
–ó–Ω–∞—á–µ–Ω–∏–µ H –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è –ø–æ –æ–¥–Ω–æ–π –∏–∑ 5 —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —Ñ–æ—Ä–º—É–ª:  
(a) –ï—Å–ª–∏ MAX = MIN: H = 0  
(b) –ï—Å–ª–∏ MAX = R –∏ G ‚â• B: H = 60.0 * (G - B) / (MAX - MIN)  
(c) –ï—Å–ª–∏ MAX = R –∏ G < B: H = 60.0 * (G - B) / (MAX - MIN) + 360.0  
(d) –ï—Å–ª–∏ MAX = G: H = 60.0 * (B - R) / (MAX - MIN) + 120.0  
(e) –ï—Å–ª–∏ MAX = B: H = 60.0 * (R - G) / (MAX - MIN) + 240.0  
–ó–Ω–∞—á–µ–Ω–∏–µ S –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º: –µ—Å–ª–∏ MAX = 0, —Ç–æ S = 0, –∏–Ω–∞—á–µ S = 1 - MIN/MAX.  
–ó–Ω–∞—á–µ–Ω–∏–µ V —Ä–∞–≤–Ω–æ MAX.
